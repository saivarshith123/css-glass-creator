## Features

* **Live Preview:** See the glassmorphism effect update in real-time as you adjust the settings.
* **Customizable Settings:**
    * Adjust **Blur** level with a slider.
    * Control **Transparency** (alpha) with a slider.
    * Pick any **Color** for the glass effect using a color picker.
    * Set an **Outline** width with a slider.
* **CSS Code Generation:** Automatically generates the cross-browser CSS for you in a text area.
* **Copy to Clipboard:** A one-click button to copy the generated code to your clipboard.

## Technologies Used

* **HTML5**
* **CSS3** (featuring `backdrop-filter` and custom gradients)
* **Vanilla JavaScript** (to handle slider inputs and generate code)

## How to Use

1.  Clone or download the repository.
2.  Open the `index.html` file in your web browser.
3.  Use the sliders and color picker to design your glass effect.
4.  Copy the generated CSS from the text box and paste it into your own project.
